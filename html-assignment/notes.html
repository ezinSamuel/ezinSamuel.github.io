HTML Tags and Descriptions

h1 to h6
Defines HTML Headings, with h1 as the largest and h6 as the smallest.
p
Defines a paragraph of text.
img
Displays an image using the "src" attribute.
a
Creates a hyperlink to another webpage or location.
ul and li
Used to create an unordered list with bullet points.
